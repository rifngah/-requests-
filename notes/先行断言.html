1.5 先行断言

先行断言

取反先行断言

a = re.findall(r'n(?!e)','final')
b = re.findall(r'n(?!e)','python')

print(a)
print(b)

['n']
['n']

正向先行断言

a = re.findall(r'n(?=e)','final')
b = re.findall(r'n(?=e)','python')
c = re.findall(r'n(?=e)','jasmine')

print(a)
print(b)
print(c)

[]
[]
['n']
